# Paroisse Saint Joseph Mokambo - Site Web

Site web officiel de la Paroisse Saint Joseph Mokambo, une communauté catholique en ligne.

## 🚀 Fonctionnalités

### Interface Utilisateur
- **Communiqué du jour** - Annonces et actualités de la paroisse
- **Temps liturgique** - Affichage automatique du temps liturgique
- **Lectures saintes** - Lectures bibliques du jour
- **Saints du jour** - Saints célébrés quotidiennement
- **Bréviaire** - Prières et offices quotidiens
- **Explication de l'Évangile** - Commentaire de l'Évangile du jour
- **Magasin paroissial** - Articles religieux disponibles

### Interface Administrateur
- **Gestion des communiqués** - Publier et gérer les annonces
- **Gestion du magasin** - Ajouter et modifier les articles
- **Tableau de bord** - Statistiques et aperçu

## 🛠️ Technologies Utilisées

- HTML5
- CSS3 (Animations, Flexbox, Grid)
- JavaScript (ES6+)
- LocalStorage pour la persistance des données
- API Catholic pour les données liturgiques
- IA gratuite pour les contenus automatiques
